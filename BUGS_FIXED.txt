   ✅ ИСПРАВЛЕНО:
   - Добавлен обработчик события click для всех .reg-plan элементов
   - При клике добавляется класс 'selected' для визуального выделения
   - Radio button автоматически отмечается
   - Начальный выбор установлен на "Стандарт"
   
   Файл: register.js (строки 150-165)
   Код:
   ```javascript
   document.querySelectorAll('.reg-plan').forEach(plan => {
     plan.addEventListener('click', (e) => {
       document.querySelectorAll('.reg-plan').forEach(p => {
         p.classList.remove('selected');
       });
       plan.classList.add('selected');
       const radio = plan.querySelector('input[type="radio"]');
       if (radio) radio.checked = true;
     });
   });
   document.getElementById('planStandard').classList.add('selected');
   ```
