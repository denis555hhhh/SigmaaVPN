📊 БАЗА ДАННЫХ:
═════════════════════════════════════════════════════════════════════════════

Таблица users:
  - id (PRIMARY KEY)
  - email (UNIQUE)
  - password (SHA256)
  - username
  - created_at
  - updated_at

Таблица subscriptions:
  - id (PRIMARY KEY)
  - user_id (FOREIGN KEY)
  - plan (Базовый/Стандарт/Премиум)
  - status (active/expired)
  - start_date
  - end_date

Таблица logs:
  - id (PRIMARY KEY)
  - user_id (FOREIGN KEY)
  - action (login/register/subscription_created)
  - details
  - created_at
