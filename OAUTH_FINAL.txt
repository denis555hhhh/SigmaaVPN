⚡ БЫСТРЫЙ СТАРТ (2 ВАРИАНТА):
═════════════════════════════════════════════════════════════════════════════

ВАРИАНТ 1: ИСПОЛЬЗОВАТЬ ДЕМО (ДЛЯ ТЕСТИРОВАНИЯ)
───────────────────────────────────────────────

1. Запустите: START_WITH_GOOGLE.bat
2. Откройте: http://localhost:8080/register.html
3. Нажмите: "Войти через Google"
4. Готово! 🎉

(Демо Client ID уже установлен в oauth_config.json)

ВАРИАНТ 2: ИСПОЛЬЗОВАТЬ СВОЙ CLIENT ID (ДЛЯ PRODUCTION)
───────────────────────────────────────────────────────

1. Запустите: RUN_OAUTH_SETUP.bat
2. Выберите: 2 (ввести свой Client ID)
3. Получите Google Client ID:
   - Откройте: https://console.cloud.google.com/
   - Создайте проект "SigmaVPN"
   - Включите Google+ API
   - Создайте OAuth 2.0 Client ID (Web)
   - Скопируйте Client ID
4. Введите Client ID в скрипт
5. Запустите: START_WITH_GOOGLE.bat
6. Откройте: http://localhost:8080/register.html
7. Нажмите: "Войти через Google"
8. Готово! 🎉
