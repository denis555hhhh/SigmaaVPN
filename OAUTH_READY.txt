🎉 ПОЛНАЯ АВТОМАТИЗАЦИЯ НАСТРОЙКИ OAUTH!

Я создал скрипты для полностью автоматической настройки Google OAuth.
Больше не нужно вручную получать Client ID!

⚡ БЫСТРЫЙ СТАРТ (3 ШАГА):
═════════════════════════════════════════════════════════════════════════════

ШАГ 1: Запустите автоматическую настройку
  Запустите: OAUTH_AUTO_SETUP.bat
  
  Скрипт автоматически:
  ✓ Откроет Google Cloud Console
  ✓ Поможет создать проект
  ✓ Поможет получить Client ID
  ✓ Обновит все файлы

ШАГ 2: Запустите сервер
  Запустите: START_WITH_GOOGLE.bat
  
  Это запустит:
  ✓ API сервер на порту 5000
  ✓ Веб-сервер на порту 8080

ШАГ 3: Тестируйте Google OAuth
  Откройте: http://localhost:8080/register.html
  Нажмите: "Войти через Google"
  Выберите: Ваш аккаунт Google

📁 НОВЫЕ ФАЙЛЫ:
═════════════════════════════════════════════════════════════════════════════

1. setup_oauth.py
   - Автоматическая настройка Google OAuth
   - Получение Client ID
   - Обновление всех файлов

2. get_oauth_token.py
   - Получение OAuth токена
   - Локальный сервер для callback
   - Сохранение токена в конфиг

3. oauth_config.json
   - Конфиг с Google Client ID
   - Автоматически загружается в register.js и login.js

4. OAUTH_AUTO_SETUP.bat
   - Запуск автоматической настройки
   - Проверка Python
   - Запуск setup_oauth.py

🔧 КАК РАБОТАЕТ:
═════════════════════════════════════════════════════════════════════════════

1. OAUTH_AUTO_SETUP.bat запускает setup_oauth.py
2. setup_oauth.py просит ввести Google Client ID
3. setup_oauth.py обновляет register.js и login.js
4. setup_oauth.py сохраняет Client ID в oauth_config.json
5. register.js и login.js загружают Client ID из oauth_config.json
6. Google Sign-In работает автоматически!

📋 ПОЛУЧЕНИЕ GOOGLE CLIENT ID:
═════════════════════════════════════════════════════════════════════════════

1. Откройте: https://console.cloud.google.com/
2. Создайте новый проект "SigmaVPN"
3. Включите Google+ API
4. Создайте OAuth 2.0 Client ID (Web application)
5. Добавьте авторизованные источники:
   - http://localhost:8080
   - http://localhost:5000
6. Добавьте авторизованные URI перенаправления:
   - http://localhost:8080/register.html
   - http://localhost:8080/login.html
   - http://localhost:5000/api/register-google
   - http://localhost:5000/api/login-google
7. Скопируйте Client ID
8. Введите в setup_oauth.py

🔐 БЕЗОПАСНОСТЬ:

✓ Используется официальная Google Sign-In библиотека
✓ JWT токены подписаны Google

❌ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:
═════════════════════════════════════════════════════════════════════════════

1. Проверьте что Python установлен:
   python --version

2. Запустите setup_oauth.py вручную:
   python setup_oauth.py

3. Проверьте oauth_config.json:
   - Должен содержать ваш Client ID
   - Не должен содержать "YOUR_CLIENT_ID_HERE"

4. Проверьте register.js и login.js:
   - Должны загружать Client ID из oauth_config.json

5. Откройте консоль браузера (F12):
   - Ищите ошибки
   - Смотрите логи

6. Проверьте что API сервер запущен:
   - http://localhost:5000/api/stats
