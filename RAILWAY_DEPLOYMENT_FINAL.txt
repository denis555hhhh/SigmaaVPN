                          🆘 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

PROBLEM: "fatal: not a git repository"
SOLUTION: Run "git init" in your project directory

PROBLEM: "fatal: 'origin' does not appear to be a 'git' repository"
SOLUTION: Run "git remote add origin https://github.com/deni555hhhh/sigma.git"

PROBLEM: "Permission denied (publickey)"
SOLUTION: Use GitHub personal access token instead of password

PROBLEM: Railway deployment fails
SOLUTION: Check Railway logs for errors, ensure Procfile and requirements.txt exist

PROBLEM: API not responding after deployment
SOLUTION: Update API_URL in JavaScript files to your Railway URL
