                              СОЗДАННЫЕ ФАЙЛЫ
═══════════════════════════════════════════════════════════════════════════════

1. configurator.html
   - Полная HTML структура конфигуратора
   - Левая колонка: параметры (устройства, страны, скорость, функции, период)
   - Правая колонка: превью карточка с расчетом цены
   - Интеграция с main.js и configurator.js

2. configurator.js (НОВЫЙ)
   - Полная логика расчета цены
   - Обработчики событий для всех элементов управления
   - Динамическое обновление превью при изменении параметров
   - Система скидок за период подписки

3. style.css (ОБНОВЛЕН)
   - Полное оформление конфигуратора
   - Адаптивный дизайн (мобильный, планшет, десктоп)
   - Стили для слайдеров, кнопок, чекбоксов
   - Зеленая цветовая схема (#16a34a)

ПАРАМЕТРЫ КОНФИГУРАЦИИ:
✓ Количество устройств: 1-10 (слайдер)
✓ Количество стран: 1-100 (слайдер)
✓ Скорость: 100 Мбит/с, 1 Гбит/с, 10 Гбит/с (кнопки)
✓ Функции: Kill Switch, Выделенный IP, Обфускация, Блокировка рекламы, Поддержка 24/7
✓ Период: 1 месяц, 3 месяца, 12 месяцев

РАСЧЕТ ЦЕНЫ:
✓ Базовая цена зависит от скорости:
  - 100 Мбит/с: 99 ₽/мес
  - 1 Гбит/с: 199 ₽/мес
  - 10 Гбит/с: 349 ₽/мес

✓ Дополнительные функции:
  - Kill Switch: включен в базовую цену
  - Выделенный IP: +50 ₽/мес
  - Обфускация трафика: +30 ₽/мес
  - Блокировка рекламы: +20 ₽/мес
  - Поддержка 24/7: +40 ₽/мес

✓ Скидки за период:
  - 1 месяц: 0% скидка
  - 3 месяца: 5% скидка
  - 12 месяцев: 20% скидка

ПРЕВЬЮ КАРТОЧКА:
✓ Отображение выбранных параметров
✓ Список включенных функций
✓ Расчет базовой цены
✓ Расчет цены функций
✓ Цена в месяц
✓ Итоговая цена за период (с учетом скидки)
✓ Кнопка "Купить подписку"

                            КАК ИСПОЛЬЗОВАТЬ
═══════════════════════════════════════════════════════════════════════════════

1. Откройте configurator.html в браузере
2. Используйте слайдеры для выбора количества устройств и стран
3. Нажимайте кнопки для выбора скорости
4. Отмечайте нужные функции чекбоксами
5. Выбирайте период подписки
6. Смотрите расчет цены в правой колонке
7. Нажимайте "Купить подписку" для оформления
