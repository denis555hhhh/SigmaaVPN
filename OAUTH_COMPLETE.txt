📋 ПОЛУЧЕНИЕ GOOGLE CLIENT ID:
═════════════════════════════════════════════════════════════════════════════

1. Откройте: https://console.cloud.google.com/
2. Создайте новый проект "SigmaVPN"
3. Включите Google+ API
4. Создайте OAuth 2.0 Client ID (Web application)
5. Добавьте авторизованные источники:
   - http://localhost:8080
   - http://localhost:5000
6. Добавьте авторизованные URI перенаправления:
   - http://localhost:8080/register.html
   - http://localhost:8080/login.html
   - http://localhost:5000/api/register-google
   - http://localhost:5000/api/login-google
7. Скопируйте Client ID
8. Введите в setup_oauth.py
