❌ ОШИБКА:
═════════════════════════════════════════════════════════════════════════════

"no registered origin"
"Ошибка 401: invalid_client"

Причина: Демо Client ID не работает с Google OAuth

✅ РЕШЕНИЕ (5 МИНУТ):
═════════════════════════════════════════════════════════════════════════════

1. Запустите: FIX_OAUTH_ERROR.bat

2. Следуйте инструкциям в скрипте

3. Введите ваш Google Client ID

4. Скрипт автоматически обновит все файлы

5. Закройте браузер

6. Запустите: START_WITH_GOOGLE.bat

7. Откройте: http://localhost:8080/register.html

8. Нажмите: "Войти через Google"

9. Готово! 🎉

📋 ПОЛУЧЕНИЕ GOOGLE CLIENT ID:
═════════════════════════════════════════════════════════════════════════════

1. Откройте: https://console.cloud.google.com/
2. Создайте проект "SigmaVPN"
3. Включите Google+ API
4. Создайте OAuth 2.0 Client ID (Web)
5. Добавьте авторизованные источники:
   - http://localhost:8080
   - http://localhost:5000
6. Добавьте авторизованные URI:
   - http://localhost:8080/register.html
   - http://localhost:8080/login.html
   - http://localhost:5000/api/register-google
   - http://localhost:5000/api/login-google
7. Скопируйте Client ID
