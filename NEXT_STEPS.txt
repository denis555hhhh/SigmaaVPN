🎯 ПРИОРИТЕТ 4: СИНХРОНИЗАЦИЯ БД С RAILWAY (ОПЦИОНАЛЬНО)
═══════════════════════════════════════════════════════════════════════════════

Если вы хотите синхронизировать БД с Railway:

1. Получите DATABASE_URL из Railway Dashboard:
   • Откройте Railway Dashboard
   • Выберите проект: sigmaVPNN
   • Перейдите в PostgreSQL сервис
   • Скопируйте DATABASE_URL

2. Добавьте DATABASE_URL в .env файл:
   DATABASE_URL=postgresql://...

3. Запустите синхронизацию:
   python download_db_railway.py

4. Проверьте, что БД синхронизирована:
   • Откройте админ-панель
   • Проверьте, что данные загружаются из Railway

✅ РЕЗУЛЬТАТ:
   • БД синхронизирована с Railway
   • Админ-панель использует данные из Railway PostgreSQL
   • Все данные актуальны

═══════════════════════════════════════════════════════════════════════════════
⚠️  ВАЖНЫЕ ЗАМЕЧАНИЯ
═══════════════════════════════════════════════════════════════════════════════

1. БУФЕР POWERSHELL
   ❌ НЕ используйте PowerShell для запуска скриптов
   ✅ Используйте Command Prompt (cmd.exe) или START.bat

2. ПОРТ 5000
   • Убедитесь, что порт 5000 не занят другим приложением
   • Если порт занят, измените PORT в .env файле

3. RAILWAY РАЗВЕРТЫВАНИЕ
   • Развертывание может занять 5-10 минут
   • Проверяйте статус на Railway Dashboard
   • Если развертывание не удалось, проверьте логи

4. СИНХРОНИЗАЦИЯ БД
   • Требует DATABASE_URL или переменные PGHOST, PGPORT, PGUSER, PGPASSWORD
   • Без этого используется локальная БД
