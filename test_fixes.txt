1. Запустите START_HERE.bat
2. Откройте браузер: http://localhost:8080
3. Откройте DevTools (F12)
4. Перейдите на вкладку Console для просмотра ошибок

🧪 ТЕСТ 1: ПЕРЕКЛЮЧЕНИЕ ТАРИФОВ
═════════════════════════════════════════════════════════════════════════════

Шаги:
  1. Откройте http://localhost:8080/register.html
  2. Заполните форму шага 1:
     - Имя: TestUser
     - Email: test1@example.com
     - Пароль: TestPassword123
     - Повторите пароль: TestPassword123
  3. Нажмите "Продолжить →"
  4. Вы должны увидеть 3 тарифа

Проверка:
  ✓ Нажмите на "Базовый"
    - Должна появиться зелёная рамка
    - Должно быть написано "Базовый"
  
  ✓ Нажмите на "Стандарт"
    - Зелёная рамка должна переместиться
    - Должно быть написано "Стандарт"
  
  ✓ Нажмите на "Премиум"
    - Зелёная рамка должна переместиться
    - Должно быть написано "Премиум"

Результат:
  [ ] Переключение работает
  [ ] Визуально выделяется выбранный тариф
  [ ] Нет ошибок в консоли
