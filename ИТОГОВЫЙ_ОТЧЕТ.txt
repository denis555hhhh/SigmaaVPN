ШАГ 1: Создать GitHub репозиторий
   → https://github.com/new
   → Имя: sigma
   → Public
   → Create

ШАГ 2: Отправить код на GitHub
   → Запустить ПОЛНАЯ_АВТОМАТИЗАЦИЯ.bat
   → Батник сделает все автоматически

ШАГ 3: Развернуть на Railway
   → https://railway.app
   → New Project
   → Deploy from GitHub
   → Выбрать sigma
   → Deploy

ШАГ 4: Получить Railway URL
   → Дождаться развертывания (2-5 минут)
   → Скопировать URL из Domains

ШАГ 5: Обновить API URLs
   → Запустить ПОЛНАЯ_АВТОМАТИЗАЦИЯ.bat
   → Выбрать опцию обновления
   → Ввести Railway URL

ШАГ 6: Финальная отправка
   → Запустить ПОЛНАЯ_АВТОМАТИЗАЦИЯ.bat
   → Выбрать опцию отправки
   → Railway перестроит приложение
