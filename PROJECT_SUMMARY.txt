🌐 КОМПОНЕНТ 2: ЛОКАЛЬНЫЙ ХОСТИНГ

Статус: ✅ Завершено

Файлы:
  • start-server.bat    - Запуск локального сервера
  • cloudflared.exe     - Cloudflare Tunnel для публичного доступа

Особенности:
  ✓ Python HTTP сервер на порту 8080
  ✓ Cloudflare Tunnel для публичного доступа
  ✓ Автоматическое создание временного домена
  ✓ Готово к использованию

Как использовать:
  1. Дважды кликните на start-server.bat
  2. Сайт будет доступен на http://localhost:8080
  3. Получите публичный URL через Cloudflare Tunnel

🚀 БЫСТРЫЙ СТАРТ

1. Запуск веб-сайта:
   → Дважды кликните на start-server.bat
   → Откройте http://localhost:8080

2. Запуск приложения:
   → Дважды кликните на app/Запустить SigmaVPN.bat
   → Или запустите app/dist/SigmaVPN.exe

3. Получение публичного доступа:
   → Используйте Cloudflare Tunnel (в start-server.bat)
   → Получите временный домен trycloudflare.com
