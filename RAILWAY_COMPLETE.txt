                            БЫСТРЫЙ СТАРТ
═══════════════════════════════════════════════════════════════════════════════

1️⃣ Создайте GitHub репозиторий
   https://github.com/new → sigmavpn-website

2️⃣ Инициализируйте Git
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/sigmavpn-website.git
   git push -u origin main

3️⃣ Создайте аккаунт на Railway
   https://railway.app

4️⃣ Создайте проект на Railway
   New Project → Deploy from GitHub → sigmavpn-website → Deploy

5️⃣ Ждите развертывания
   2-5 минут

6️⃣ Откройте URL вашего сайта
   https://sigmavpn-website.railway.app

                            РЕШЕНИЕ ПРОБЛЕМ
═══════════════════════════════════════════════════════════════════════════════

❌ Приложение не запускается
✅ Проверьте логи: Railway → Logs
✅ Убедитесь, что requirements.txt содержит все зависимости
✅ Проверьте синтаксис Procfile

❌ API не работает
✅ Обновите API_URL в JavaScript файлах
✅ Проверьте CORS настройки в app.py
✅ Проверьте логи ошибок

❌ База данных не создается
✅ app.py автоматически создает базу при первом запуске
✅ Проверьте права доступа

❌ Статические файлы не загружаются
✅ Убедитесь, что HTML, CSS, JS файлы в корневой папке
✅ Проверьте пути в app.py
