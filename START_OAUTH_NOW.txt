⚡ САМЫЙ БЫСТРЫЙ СПОСОБ (5 МИНУТ):
═════════════════════════════════════════════════════════════════════════════

1. Откройте Google Cloud Console:
   https://console.cloud.google.com/

2. Создайте проект "SigmaVPN" и получите Client ID
   (Следуйте инструкциям в QUICK_OAUTH_SETUP.txt)

3. Запустите OAUTH_AUTO_SETUP.bat

4. Введите ваш Client ID

5. Запустите START_WITH_GOOGLE.bat

6. Откройте http://localhost:8080/register.html

7. Нажмите "Войти через Google"

8. Готово! 🎉
