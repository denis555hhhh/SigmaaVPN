                            ПОСЛЕ РАЗВЕРТЫВАНИЯ
═══════════════════════════════════════════════════════════════════════════════

1. ОБНОВИТЕ API URL

   Откройте файлы:
   - register.js
   - login.js
   - checkout.js
   - cabinet.js

   Найдите:
   const API_URL = 'http://localhost:5000';

   Замените на:
   const API_URL = 'https://your-railway-app.railway.app';

2. ЗАКОММИТЬТЕ И ОТПРАВЬТЕ

   git add .
   git commit -m "Update API URL for Railway"
   git push origin main

3. RAILWAY ПЕРЕСТРОИТ ПРИЛОЖЕНИЕ

Railway автоматически устанавливает:
- PORT: 5000 (или другой доступный порт)

- DEBUG: false

                            РЕШЕНИЕ ПРОБЛЕМ
═══════════════════════════════════════════════════════════════════════════════

❌ Приложение не запускается
✅ Проверьте логи в Railway → Logs
✅ Убедитесь, что requirements.txt содержит все зависимости
✅ Проверьте Procfile синтаксис

❌ API не работает
✅ Обновите API_URL в JavaScript файлах
✅ Проверьте CORS настройки в app.py
✅ Проверьте логи ошибок

❌ База данных не создается
✅ app.py автоматически создает базу при первом запуске
✅ Проверьте права доступа к папке

❌ Статические файлы не загружаются
✅ Убедитесь, что HTML, CSS, JS файлы в корневой папке
✅ Проверьте пути в app.py
