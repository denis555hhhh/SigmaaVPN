📋 БЫСТРАЯ ИНСТРУКЦИЯ
═══════════════════════════════════════════════════════════════════════════════

Шаг 1: Получить DATABASE_URL
  1. Откройте https://railway.app
  2. Выберите проект SigmaVPN
  3. Перейдите в Data > PostgreSQL
  4. Найдите CONNECTION STRING (DATABASE_URL)
  5. Скопируйте строку

Шаг 2: Создать .env файл
  1. Откройте Notepad
  2. Вставьте: DATABASE_URL=postgresql://...
  3. Сохраните как ".env" в папке проекта

Шаг 3: Скачать БД
  1. Откройте Command Prompt
  2. Перейдите в папку проекта
  3. Выполните: python download_db_railway.py
  4. БД будет скачана с Railway

Шаг 4: Запустить панель
  1. Выполните: python admin_server.py
  2. Откройте: http://localhost:5000
  3. Готово!
