❌ ПРИЧИНА ОШИБКИ:
═════════════════════════════════════════════════════════════════════════════

Демо Client ID не работает с Google OAuth.
Нужен реальный Client ID от Google Cloud Console.

✅ РЕШЕНИЕ (5 МИНУТ):
═════════════════════════════════════════════════════════════════════════════

СПОСОБ 1: АВТОМАТИЧЕСКОЕ ИСПРАВЛЕНИЕ (РЕКОМЕНДУЕТСЯ)
───────────────────────────────────────────────────

1. Запустите: FIX_OAUTH_ERROR.bat
2. Следуйте инструкциям в скрипте
3. Введите ваш Google Client ID
4. Скрипт автоматически обновит все файлы
5. Готово! 🎉

СПОСОБ 2: РУЧНОЕ ИСПРАВЛЕНИЕ
──────────────────────────────

1. Получите Google Client ID:
   - Откройте: https://console.cloud.google.com/
   - Создайте проект "SigmaVPN"
   - Включите Google+ API
   - Создайте OAuth 2.0 Client ID (Web)
   - Добавьте авторизованные источники:
     * http://localhost:8080
     * http://localhost:5000
   - Добавьте авторизованные URI:
     * http://localhost:8080/register.html
     * http://localhost:8080/login.html
     * http://localhost:5000/api/register-google
     * http://localhost:5000/api/login-google
   - Скопируйте Client ID

2. Обновите oauth_config.json:
   - Откройте oauth_config.json
   - Замените "google_client_id" на ваш Client ID
   - Сохраните файл

3. Готово! 🎉

📋 ДЕТАЛЬНЫЕ ШАГИ ДЛЯ ПОЛУЧЕНИЯ CLIENT ID:
═════════════════════════════════════════════════════════════════════════════

ШАГ 1: ОТКРОЙТЕ GOOGLE CLOUD CONSOLE
  https://console.cloud.google.com/

ШАГ 2: СОЗДАЙТЕ НОВЫЙ ПРОЕКТ
  - Нажмите на выпадающее меню проектов (вверху)
  - Нажмите "Новый проект"
  - Введите название: SigmaVPN
  - Нажмите "Создать"
  - Подождите 1-2 минуты

ШАГ 3: ВКЛЮЧИТЕ GOOGLE+ API
  - Перейдите в "APIs & Services" → "Library"
  - Найдите "Google+ API"
  - Нажмите "Включить"

ШАГ 4: СОЗДАЙТЕ OAUTH 2.0 CLIENT ID
  - Перейдите в "APIs & Services" → "Credentials"
  - Нажмите "Создать учетные данные" → "OAuth 2.0 Client ID"
  - Выберите "Веб-приложение"
  - Введите название: SigmaVPN
  - Нажмите "Создать"

ШАГ 5: ДОБАВЬТЕ АВТОРИЗОВАННЫЕ ИСТОЧНИКИ
  - Нажмите на созданное приложение
  - Найдите "Авторизованные источники JavaScript"
  - Добавьте:
    * http://localhost:8080
    * http://localhost:5000

ШАГ 6: ДОБАВЬТЕ АВТОРИЗОВАННЫЕ URI ПЕРЕНАПРАВЛЕНИЯ
  - Найдите "Авторизованные URI перенаправления"
  - Добавьте:
    * http://localhost:8080/register.html
    * http://localhost:8080/login.html
    * http://localhost:5000/api/register-google
    * http://localhost:5000/api/login-google
  - Нажмите "Сохранить"

ШАГ 7: СКОПИРУЙТЕ CLIENT ID
  - Нажмите на приложение
  - Скопируйте "Client ID" (длинная строка)
  - Пример: 123456789-abcdefghijklmnopqrstuvwxyz.apps.googleusercontent.com
