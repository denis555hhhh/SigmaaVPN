🚀 ВЕРСИИ И СПОСОБЫ ЗАПУСКА
═══════════════════════════════════════════════════════════════════════════════

ВЕРСИЯ 1: БАЗОВАЯ (START.bat)
   Для: Новичков, простых задач
   Запуск: Дважды кликните START.bat
   Функции: Админ-панель, статистика, пользователи, платежи, подписки

ВЕРСИЯ 2: РАСШИРЕННАЯ (START_EXTENDED.bat)
   Для: Локального управления
   Запуск: Дважды кликните START_EXTENDED.bat
   Функции: Все из базовой + логи, сообщения, покупки, поиск

ВЕРСИЯ 3: PRO (START_PRO.bat)
   Для: Production, Railway, полного управления
   Запуск: Дважды кликните START_PRO.bat
   Функции: Все из расширенной + Railway, логирование, экспорт

ВЕРСИЯ 4: EXE (SigmaVPN Admin Panel.exe) ⭐ РЕКОМЕНДУЕТСЯ
   Для: Standalone приложение, распространение
   Запуск: Дважды кликните SigmaVPN Admin Panel.exe
   Функции: Все из Pro + Telegram бот, не требует Python
