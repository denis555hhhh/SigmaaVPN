🚀 КАК РАЗВЕРНУТЬ НА RAILWAY
═══════════════════════════════════════════════════════════════════════════════

Способ 1: Через Railway Dashboard (РЕКОМЕНДУЕТСЯ)
───────────────────────────────────────────────────────────────────────────────

1. Откройте https://railway.app
2. Выберите проект SigmaVPN
3. Нажмите "+ New Service"
4. Выберите "GitHub Repo"
5. Выберите репозиторий sigmaVPNN
6. Нажмите "Deploy"
7. Перейдите в Variables и добавьте:
   ├─ PORT=5000
   └─ DATABASE_PATH=sigmavpn.db
8. Готово! Панель будет доступна по URL


Способ 2: Через Git Push
───────────────────────────────────────────────────────────────────────────────

1. Обновите Procfile:
   web: python admin_server.py

2. Закоммитьте и запушьте:
   git add Procfile
   git commit -m "Deploy admin panel"
   git push origin main

3. Railway автоматически развернет
