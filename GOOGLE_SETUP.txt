⚡ БЫСТРАЯ НАСТРОЙКА (5 МИНУТ):
═════════════════════════════════════════════════════════════════════════════

1. Откройте Google Cloud Console:
   https://console.cloud.google.com/

2. Создайте новый проект:
   - Нажмите на выпадающее меню проектов (вверху)
   - Нажмите "Новый проект"
   - Введите название: "SigmaVPN"
   - Нажмите "Создать"

3. Включите Google+ API:
   - Перейдите в "APIs & Services" → "Library"
   - Найдите "Google+ API"
   - Нажмите "Включить"

4. Создайте OAuth 2.0 учетные данные:
   - Перейдите в "APIs & Services" → "Credentials"
   - Нажмите "Создать учетные данные" → "OAuth 2.0 Client ID"
   - Выберите "Веб-приложение"
   - Добавьте авторизованные источники JavaScript:
     * http://localhost:8080
     * http://localhost:5000
   - Добавьте авторизованные URI перенаправления:
     * http://localhost:8080/register.html
     * http://localhost:8080/login.html
     * http://localhost:5000/api/register-google
     * http://localhost:5000/api/login-google
   - Нажмите "Создать"

5. Скопируйте Client ID:
   - Вы увидите окно с Client ID
   - Скопируйте значение Client ID

6. Обновите файлы:
   - Откройте register.js
   - Найдите строку с '1234567890-abcdefghijklmnopqrstuvwxyz.apps.googleusercontent.com'
   - Замените на ваш Client ID
   - Сделайте то же самое в login.js

7. Готово!
   - Теперь Google Sign-In будет работать

🔍 ГДЕ НАЙТИ CLIENT ID:
═════════════════════════════════════════════════════════════════════════════

1. Google Cloud Console → APIs & Services → Credentials
2. Найдите "OAuth 2.0 Client IDs"
3. Нажмите на "Web client"
4. Скопируйте "Client ID"

🚀 ТЕСТИРОВАНИЕ:
═════════════════════════════════════════════════════════════════════════════

1. Запустите simple_start.bat
2. Откройте http://localhost:8080/register.html
3. Нажмите кнопку "Войти через Google"
4. Выберите аккаунт Google
5. Должна произойти регистрация

✅ УСПЕШНАЯ РЕГИСТРАЦИЯ:
═════════════════════════════════════════════════════════════════════════════

Если вы видите:
- Окно выбора аккаунта Google
- Перенаправление на главную страницу
- Сообщение "Добро пожаловать!"

То Google Sign-In работает правильно!

❌ ТИПИЧНЫЕ ОШИБКИ:
═════════════════════════════════════════════════════════════════════════════

ОШИБКА: "Ошибка: Unexpected token < in JSON at position 0"
  Причина: Client ID не установлен или неправильный
  Решение: Проверьте Client ID в register.js и login.js

ОШИБКА: "Ошибка подключения"
  Причина: API сервер не запущен
  Решение: Запустите python app.py

ОШИБКА: "Пользователь не найден"
  Причина: Пользователь с этим email не зарегистрирован
  Решение: Сначала зарегистрируйтесь через Google
