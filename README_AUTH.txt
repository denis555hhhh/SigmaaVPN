🚀 КАК ИСПОЛЬЗОВАТЬ:
═════════════════════════════════════════════════════════════════════════════

ШАГИ:

1. Запустите START_HERE.bat
   - Это запустит API сервер и веб-сервер

2. Откройте браузер: http://localhost:8080

3. Нажмите "Зарегистрироваться" или перейдите на:
   http://localhost:8080/register.html

4. Заполните форму регистрации:
   - Имя: Любое имя
   - Email: Ваш email (должен быть уникальным)
   - Пароль: Минимум 8 символов
   - Повторите пароль
   - Выберите тариф
   - Согласитесь с условиями

5. После регистрации вы будете перенаправлены на главную страницу

6. Для входа перейдите на: http://localhost:8080/login.html
   - Email: Ваш email
   - Пароль: Ваш пароль
   - Опционально: Отметьте "Запомнить меня"

📊 ТЕСТИРОВАНИЕ:
═════════════════════════════════════════════════════════════════════════════

Для тестирования API используйте:

1. Запустите test_api.py:
   python test_api.py

   Это автоматически:
   - Создаст тестового пользователя
   - Проверит вход
   - Создаст подписку
   - Получит профиль
   - Выведет статистику

2. Или используйте curl/Postman:

   Регистрация:
   POST http://localhost:5000/api/register
   {
     "username": "TestUser",
     "email": "test@example.com",
     "password": "TestPassword123"
   }

   Вход:
   POST http://localhost:5000/api/login
   {
     "email": "test@example.com",
     "password": "TestPassword123"
   }

   Профиль:
   GET http://localhost:5000/api/profile/1

   Подписка:
   POST http://localhost:5000/api/subscription
   {
     "user_id": 1,
     "plan": "Стандарт"
   }

💾 ДАННЫЕ В БРАУЗЕРЕ:
═════════════════════════════════════════════════════════════════════════════

После регистрации/входа в localStorage сохраняются:
- user_id: ID пользователя
- username: Имя пользователя
- email: Email пользователя
- subscription_id: ID подписки
- plan: Тариф подписки
- is_logged_in: Статус входа
- remember_me: Запомнить меня

Вы можете проверить в DevTools (F12 → Application → Local Storage)
