1. ДЕСКТОПНОЕ ПРИЛОЖЕНИЕ (db_launcher.py)

   ├─ Порт: Локальный

      └─ Настройки БД

2. ВЕБ-ПРИЛОЖЕНИЕ (admin_panel.py)
   ├─ Фреймворк: Flask
   ├─ Порт: 5001
   ├─ Зависимости: flask, flask-cors
   └─ Функции:
      ├─ Аутентификация
      ├─ REST API
      ├─ HTML интерфейс
      ├─ Управление пользователями
      ├─ Управление подписками
      ├─ Просмотр логов
      ├─ Статистика
      └─ Управление БД

   ├─ Тип: SQLite3
   ├─ Таблицы: 3
   └─ Структура:
      ├─ users (id, email, password, username, created_at, updated_at)
      ├─ subscriptions (id, user_id, plan, status, start_date, end_date)
      └─ logs (id, user_id, action, details, created_at)

🗄️ СТРУКТУРА БД:
════════════════════════════════════════════════════════════════

ТАБЛИЦА: users
┌──────────────────────────────────────────┐
│ id (PK)      │ INTEGER PRIMARY KEY        │
│ email        │ TEXT UNIQUE NOT NULL       │
│ password     │ TEXT NOT NULL (SHA256)     │
│ username     │ TEXT                       │
│ created_at   │ TIMESTAMP DEFAULT NOW      │
│ updated_at   │ TIMESTAMP DEFAULT NOW      │
└──────────────────────────────────────────┘

ТАБЛИЦА: subscriptions
┌──────────────────────────────────────────┐
│ id (PK)      │ INTEGER PRIMARY KEY        │
│ user_id (FK) │ INTEGER FOREIGN KEY        │
│ plan         │ TEXT NOT NULL              │
│ status       │ TEXT DEFAULT 'active'      │
│ start_date   │ TIMESTAMP DEFAULT NOW      │
│ end_date     │ TIMESTAMP                  │
└──────────────────────────────────────────┘

ТАБЛИЦА: logs
┌──────────────────────────────────────────┐
│ id (PK)      │ INTEGER PRIMARY KEY        │
│ user_id (FK) │ INTEGER FOREIGN KEY        │
│ action       │ TEXT NOT NULL              │
│ details      │ TEXT                       │
│ created_at   │ TIMESTAMP DEFAULT NOW      │
└──────────────────────────────────────────┘

📡 API ENDPOINTS (Веб-приложение):
════════════════════════════════════════════════════════════════

АУТЕНТИФИКАЦИЯ:
├─ POST /api/admin/login          - Вход
└─ POST /api/admin/logout         - Выход

ПОЛЬЗОВАТЕЛИ:
├─ GET  /api/admin/users          - Список пользователей
├─ POST /api/admin/users          - Добавить пользователя
└─ DELETE /api/admin/users/<id>   - Удалить пользователя

ПОДПИСКИ:
├─ GET  /api/admin/subscriptions  - Список подписок
├─ POST /api/admin/subscriptions  - Добавить подписку
└─ DELETE /api/admin/subscriptions/<id> - Удалить подписку

ЛОГИ:
├─ GET  /api/admin/logs           - Список логов
└─ DELETE /api/admin/logs         - Очистить логи

СТАТИСТИКА:
├─ GET  /api/admin/stats          - Статистика
└─ GET  /api/admin/dbinfo         - Информация о БД

УПРАВЛЕНИЕ БД:
├─ POST /api/admin/backup         - Резервная копия
└─ POST /api/admin/recreate       - Пересоздать БД
