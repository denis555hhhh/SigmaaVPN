🎯 ДЕТАЛЬНЫЕ ШАГИ:
═════════════════════════════════════════════════════════════════════════════

ШАГИ 1-5: ПОЛУЧЕНИЕ GOOGLE CLIENT ID (3 МИНУТЫ)
───────────────────────────────────────────────────

1. Откройте Google Cloud Console:
   https://console.cloud.google.com/

2. Создайте новый проект:
   - Нажмите на выпадающее меню проектов (вверху)
   - Нажмите "Новый проект"
   - Введите: SigmaVPN
   - Нажмите "Создать"
   - Подождите 1-2 минуты

3. Включите Google+ API:
   - Перейдите в "APIs & Services" → "Library"
   - Найдите "Google+ API"
   - Нажмите "Включить"

4. Создайте OAuth 2.0 Client ID:
   - Перейдите в "APIs & Services" → "Credentials"
   - Нажмите "Создать учетные данные" → "OAuth 2.0 Client ID"
   - Выберите "Веб-приложение"
   - Введите название: SigmaVPN
   - Нажмите "Создать"

5. Добавьте авторизованные источники:
   - Нажмите на созданное приложение
   - Найдите "Авторизованные источники JavaScript"
   - Добавьте:
     * http://localhost:8080
     * http://localhost:5000
   - Найдите "Авторизованные URI перенаправления"
   - Добавьте:
     * http://localhost:8080/register.html
     * http://localhost:8080/login.html
     * http://localhost:5000/api/register-google
     * http://localhost:5000/api/login-google
   - Нажмите "Сохранить"

6. Скопируйте Client ID:
   - Нажмите на приложение
   - Скопируйте "Client ID" (длинная строка)

ШАГИ 6-8: АВТОМАТИЧЕСКАЯ НАСТРОЙКА (2 МИНУТЫ)
───────────────────────────────────────────────

6. Запустите OAUTH_AUTO_SETUP.bat:
   - Дважды нажмите на OAUTH_AUTO_SETUP.bat
   - Откроется командная строка

7. Введите Client ID:
   - Скопируйте Client ID из Google Cloud Console
   - Вставьте в командную строку
   - Нажмите Enter

8. Готово!
   - Скрипт автоматически обновит все файлы
   - Закройте командную строку

✅ ПРОВЕРКА:
═════════════════════════════════════════════════════════════════════════════

1. Запустите START_WITH_GOOGLE.bat
2. Откройте http://localhost:8080/register.html
3. Нажмите "Войти через Google"
4. Выберите аккаунт Google
5. Должна произойти регистрация

Если видите сообщение "Добро пожаловать!" - всё работает! 🎉

❌ ЕСЛИ НЕ РАБОТАЕТ:
═════════════════════════════════════════════════════════════════════════════

1. Проверьте что Client ID правильный:
   - Откройте oauth_config.json
   - Должен содержать ваш Client ID
   - Не должен содержать "YOUR_CLIENT_ID_HERE"

2. Проверьте что API сервер запущен:
   - Откройте http://localhost:5000/api/stats
   - Должен вернуть JSON

3. Откройте консоль браузера (F12):
   - Ищите ошибки
   - Смотрите логи

4. Проверьте авторизованные источники:
   - Откройте Google Cloud Console
   - Проверьте что добавили http://localhost:8080 и http://localhost:5000

5. Проверьте авторизованные URI:
   - Откройте Google Cloud Console
   - Проверьте что добавили все 4 URI
