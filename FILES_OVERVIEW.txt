🔧 ОСНОВНЫЕ ФАЙЛЫ ПРОЕКТА
═══════════════════════════════════════════════════════════════════════════════

admin_panel.html
   • Веб-интерфейс админ-панели
   • HTML/CSS/JavaScript
   • Красивый дизайн с анимациями
   • Разделы: Статистика, Пользователи, Платежи, Подписки
   • Использует API endpoints для загрузки данных

admin_server.py
   • Flask сервер для админ-панели
   • Python
   • Слушает на http://0.0.0.0:5000
   • 4 API endpoints: /api/users, /api/payments, /api/subscriptions, /api/stats
   • Подключается к SQLite БД (sigmavpn.db)
   • Совместим с Railway

run_admin_panel.py (НОВЫЙ)
   • Автоматический запуск админ-панели
   • Python
   • Синхронизирует БД с Railway
   • Открывает браузер автоматически
   • Красивые логи и информационные сообщения

START.bat
   • Быстрый запуск админ-панели
   • Batch скрипт для Windows
   • Проверяет Python и зависимости
   • Запускает run_admin_panel.py
   • Дважды кликните для запуска

📊 БД И КОНФИГУРАЦИЯ
═══════════════════════════════════════════════════════════════════════════════

sigmavpn.db
   • Локальная SQLite БД
   • Размер: 0.04 MB
   • Таблицы: 7 (telegram_users, telegram_payments, telegram_subscriptions и т.д.)
   • Пользователи: 3
   • Используется для локального тестирования

sigmavpn.db.backup
   • Резервная копия локальной БД
   • Создается автоматически при синхронизации

.env
   • Переменные окружения
   • DATABASE_PATH=sigmavpn.db
   • PORT=5000
   • TELEGRAM_BOT_TOKEN=...
   • YOOMONEY_WALLET=4100118775331265
   • PGHOST, PGPORT, PGUSER, PGPASSWORD, PGDATABASE (для Railway)

requirements.txt
   • Зависимости Python
   • Flask 3.0.0
   • Flask-CORS 4.0.0
   • python-dotenv 1.0.0
   • psycopg2-binary 2.9.9
   • gunicorn 21.2.0

Procfile
   • Конфигурация Railway
   • web: python admin_server.py
   • worker: python telegram_bot.py

🔄 СИНХРОНИЗАЦИЯ БД
═══════════════════════════════════════════════════════════════════════════════

download_db_railway.py
   • Синхронизация БД с Railway PostgreSQL
   • Python
   • Загружает БД с Railway в локальную SQLite
   • Требует DATABASE_URL или переменные PGHOST, PGPORT, PGUSER, PGPASSWORD
   • Создает резервную копию перед синхронизацией
   • Автоматически запускается при старте админ-панели

sync_db_from_railway.py
   • Альтернативный скрипт синхронизации
   • Python
   • Использует Railway CLI (если установлен)
