✅ GOOGLE SIGN-IN ГОТОВ К ИСПОЛЬЗОВАНИЮ!

Регистрация и вход через Google полностью реализованы.
Требуется только получить Google Client ID и обновить его в коде.

⚡ БЫСТРЫЙ СТАРТ (5 МИНУТ):
═════════════════════════════════════════════════════════════════════════════

1️⃣  ПОЛУЧИТЬ GOOGLE CLIENT ID:
   
   Откройте: https://console.cloud.google.com/
   
   - Создайте новый проект "SigmaVPN"
   - Включите Google+ API
   - Создайте OAuth 2.0 Client ID (Web application)
   - Добавьте авторизованные источники:
     * http://localhost:8080
     * http://localhost:5000
   - Скопируйте Client ID

2️⃣  ОБНОВИТЬ CLIENT ID В КОДЕ:
   
   Откройте register.js:
   - Найдите: '1234567890-abcdefghijklmnopqrstuvwxyz.apps.googleusercontent.com'
   - Замените на ваш Client ID
   
   Откройте login.js:
   - Найдите: '1234567890-abcdefghijklmnopqrstuvwxyz.apps.googleusercontent.com'
   - Замените на ваш Client ID

3️⃣  ЗАПУСТИТЬ И ТЕСТИРОВАТЬ:
   
   Запустите: START_WITH_GOOGLE.bat
   
   Откройте браузер: http://localhost:8080/register.html
   
   Нажмите: "Войти через Google"
   
   Выберите аккаунт Google и готово!
