ВЕРСИЯ 1: БАЗОВАЯ (START.bat)
─────────────────────────────────────────────────────────────────────────────

ФАЙЛЫ:
   • admin_panel.html - веб-интерфейс
   • admin_server.py - Flask сервер
   • START.bat - запуск

ФУНКЦИИ:
   • Админ-панель с 4 разделами
   • Статистика в реальном времени
   • Таблицы пользователей, платежей, подписок
   • API endpoints для данных

ИНТЕРФЕЙС:
   • Простой и понятный
   • Неоновый дизайн
   • Адаптивный для мобильных

ПРОИЗВОДИТЕЛЬНОСТЬ:
   • Быстрая загрузка
   • Минимальные требования
   • Работает на слабых ПК

ИСПОЛЬЗОВАНИЕ:
   Для локального тестирования и простых задач

   Для production, Railway, полного управления

🔄 МИГРАЦИЯ МЕЖДУ ВЕРСИЯМИ
═══════════════════════════════════════════════════════════════════════════════

БАЗОВАЯ → РАСШИРЕННАЯ:
   1. Скачайте новые файлы
   2. Запустите START_EXTENDED.bat
   3. Все данные сохранятся

РАСШИРЕННАЯ → PRO:
   1. Скачайте новые файлы
   2. Настройте Railway в .env
   3. Запустите START_PRO.bat
   4. Все данные синхронизируются

ОТКАТ:
   1. Просто запустите старый .bat файл
   2. Все данные остаются в БД
