                            ГЕНЕРАЦИЯ КЛЮЧЕЙ
═══════════════════════════════════════════════════════════════════════════════

Каждый ключ состоит из:
- Уникальный ID: key_[9 символов]_[timestamp]
- Значение ключа: 64 случайных символа (A-Z, a-z, 0-9)

Пример ключа:
key_a7f3k2m9_1716806400
ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789ABCD

                            ХРАНЕНИЕ ДАННЫХ
═══════════════════════════════════════════════════════════════════════════════

Ключи хранятся в localStorage браузера:
- Ключ: 'happ_keys'
- Формат: JSON массив объектов

Структура ключа:
{
  "id": "key_a7f3k2m9_1716806400",
  "name": "Мой ноутбук",
  "value": "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789ABCD",
  "device": "windows",
  "access": "premium",
  "regions": ["eu", "us", "asia"],
  "created": "2026-05-27T10:00:00.000Z",
  "expires": "2026-08-25T10:00:00.000Z",
  "status": "active"
}

                            КАК ИСПОЛЬЗОВАТЬ
═══════════════════════════════════════════════════════════════════════════════

1. СОЗДАНИЕ КЛЮЧА:
   - Откройте key-configurator.html
   - Введите название ключа
   - Выберите тип устройства
   - Выберите срок действия
   - Выберите уровень доступа
   - Выберите регионы доступа
   - Нажмите "Создать ключ"
   - Скопируйте ключ из модального окна

2. ПРОСМОТР КЛЮЧЕЙ:
   - Все созданные ключи отображаются в правой колонке
   - Используйте поиск для поиска по названию или ID
   - Используйте фильтр для фильтрации по устройству

3. УПРАВЛЕНИЕ КЛЮЧОМ:
   - Нажмите "Подробнее" на карточке ключа
   - Просмотрите всю информацию о ключе
   - Нажмите "Отозвать ключ" для удаления
