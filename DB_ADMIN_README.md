## 📝 Примеры использования

### Добавить пользователя
1. Откройте админ-панель
2. Перейдите в "Пользователи"
3. Нажмите "➕ Добавить пользователя"
4. Заполните форму
5. Нажмите "Сохранить"

### Создать подписку
1. Перейдите в "Подписки"
2. Нажмите "➕ Добавить подписку"
3. Выберите пользователя
4. Выберите план (Базовый/Стандарт/Премиум)
5. Нажмите "Добавить"

### Создать резервную копию
1. Перейдите в "Настройки"
2. Нажмите "💾 Резервная копия"
3. Файл будет сохранен как `sigmavpn_backup_YYYYMMDD_HHMMSS.db`

## 🐛 Решение проблем

### Ошибка: "Python не установлен"
- Установите Python с https://www.python.org
- Убедитесь, что Python добавлен в PATH

### Ошибка: "Flask не установлен"
```bash
pip install flask flask-cors
```

### БД не создается
```bash
python create_database.py
```

### Порт 5001 занят
Измените порт в `admin_panel.py`:
```python
app.run(debug=False, host='0.0.0.0', port=5002)  # Измените 5001 на 5002
```
