📋 ПОШАГОВЫЕ ИНСТРУКЦИИ
═══════════════════════════════════════════════════════════════════════════════

ШАГ 1: Откройте Command Prompt
   • Нажмите Win+R
   • Введите: cmd
   • Нажмите Enter

ШАГ 2: Перейдите в папку проекта
   • Выполните: cd "c:\Users\Gnida222\Desktop\Сайт впн"

ШАГ 3: Проверьте статус Git
   • Выполните: git status
   • Вы должны увидеть список измененных файлов

ШАГ 4: Добавьте файлы в Git
   • Выполните: git add admin_server.py START.bat run_admin_panel.py ADMIN_PANEL_STATUS.txt CHECKLIST_ADMIN_PANEL.txt FINAL_ADMIN_PANEL_REPORT.txt

ШАГ 5: Создайте коммит
   • Выполните: git commit -m "Improve admin panel: add auto-launcher, update server for Railway compatibility, add documentation"

ШАГ 6: Загрузите на GitHub
   • Выполните: git push origin main

ШАГ 7: Проверьте результат
   • Откройте https://github.com/denis555hhhh/sigmaVPNN
   • Проверьте, что коммит загружен
