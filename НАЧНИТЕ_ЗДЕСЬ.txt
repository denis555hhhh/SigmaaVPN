1. Дважды кликните на файл: ПОЛНАЯ_АВТОМАТИЗАЦИЯ.bat

   Батник автоматически:
   ✓ Проверит Git и Python
   ✓ Инициализирует Git репозиторий
   ✓ Добавит все файлы
   ✓ Создаст коммит
   ✓ Отправит код на GitHub

2. Создайте репозиторий на GitHub:
   - Перейдите на https://github.com/new
   - Имя: sigma
   - Выберите: Public
   - Нажмите "Create repository"

3. Разверните на Railway:
   - Перейдите на https://railway.app
   - New Project → Deploy from GitHub
   - Выберите репозиторий sigma
   - Нажмите Deploy
