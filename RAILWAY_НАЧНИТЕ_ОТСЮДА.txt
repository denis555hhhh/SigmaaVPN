                            6 ШАГОВ К РАЗВЕРТЫВАНИЮ
═══════════════════════════════════════════════════════════════════════════════

1️⃣ СОЗДАЙТЕ GITHUB РЕПОЗИТОРИЙ

   Откройте: https://github.com/new
   
   Заполните:
   - Repository name: sigmavpn-website
   - Public
   
   Нажмите: Create repository

2️⃣ ИНИЦИАЛИЗИРУЙТЕ GIT

   Откройте PowerShell в папке проекта:
   
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USERNAME/sigmavpn-website.git
   git push -u origin main

3️⃣ СОЗДАЙТЕ АККАУНТ НА RAILWAY

   Откройте: https://railway.app
   Sign Up → GitHub

4️⃣ СОЗДАЙТЕ ПРОЕКТ НА RAILWAY

   New Project → Deploy from GitHub
   Выберите: sigmavpn-website
   Нажмите: Deploy

5️⃣ ЖДИТЕ РАЗВЕРТЫВАНИЯ

   2-5 минут

6️⃣ ОТКРОЙТЕ ВАШ САЙТ

   Скопируйте URL из Railway
   Откройте в браузере

⚠️ ВАЖНО: Обновите API URL

Откройте файлы:
- register.js
- login.js
- checkout.js
- cabinet.js

Найдите:
const API_URL = 'http://localhost:5000';

Замените на:
const API_URL = 'https://sigmavpn-website.railway.app';

Затем:
git add .
git commit -m "Update API URL"
git push origin main
