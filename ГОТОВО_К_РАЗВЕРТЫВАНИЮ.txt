ЭТАП 1: GITHUB РЕПОЗИТОРИЙ
────────────────────────────────────────────────────────────────────────────────

1. Перейдите на https://github.com/new
2. Заполните форму:
   - Repository name: sigma
   - Description: SigmaVPN website with API
   - Visibility: Public
3. Нажмите "Create repository"

ЭТАП 2: ОТПРАВКА КОДА
────────────────────────────────────────────────────────────────────────────────

Способ 1 (Автоматический):
- Запустите НАЧНИТЕ_ОТСЮДА.bat
- Выберите опцию 1 "Отправить код на GitHub"
- Следуйте инструкциям

Способ 2 (Ручной):
Откройте Command Prompt и выполните:

cd "c:\Users\Gnida222\Desktop\Сайт впн"
git remote add origin https://github.com/ВАШ_ЮЗЕРНЕЙМ/sigma.git
git branch -M main
git push -u origin main

Замените ВАШ_ЮЗЕРНЕЙМ на ваше имя пользователя GitHub

ЭТАП 3: РАЗВЕРТЫВАНИЕ НА RAILWAY
────────────────────────────────────────────────────────────────────────────────

1. Перейдите на https://railway.app
2. Нажмите "New Project"
3. Выберите "Deploy from GitHub"
4. Авторизуйте GitHub (если требуется)
5. Выберите репозиторий "sigma"
6. Нажмите "Deploy"

Railway автоматически:
✓ Установит зависимости из requirements.txt
✓ Запустит приложение через Procfile
✓ Создаст публичный URL
✓ Настроит переменные окружения

Ожидание: 2-5 минут

ЭТАП 4: ПОЛУЧЕНИЕ RAILWAY URL
────────────────────────────────────────────────────────────────────────────────

После развертывания:
1. Перейдите на страницу проекта Railway
2. Найдите раздел "Domains"
3. Скопируйте URL (например: https://sigma-production.railway.app)

ЭТАП 5: ОБНОВЛЕНИЕ API URLs
────────────────────────────────────────────────────────────────────────────────

Способ 1 (Автоматический):
- Запустите НАЧНИТЕ_ОТСЮДА.bat
- Выберите опцию 2 "Обновить API URLs"
- Введите ваш Railway URL
- Выберите "y" для коммита и отправки

Способ 2 (Ручной):
Отредактируйте эти файлы:
- register.js
- login.js
- checkout.js
- cabinet.js

Найдите строку:
const API_URL = 'http://localhost:5000';

Замените на:
const API_URL = 'https://ВАШ_RAILWAY_URL';

ЭТАП 6: ФИНАЛЬНАЯ ОТПРАВКА
────────────────────────────────────────────────────────────────────────────────

Откройте Command Prompt и выполните:

cd "c:\Users\Gnida222\Desktop\Сайт впн"
git add .
git commit -m "Update API URLs for Railway"
git push origin main

Railway автоматически перестроит приложение!

                          ✅ ПРОВЕРКА РАБОТЫ
════════════════════════════════════════════════════════════════════════════════

После развертывания проверьте:

1. Главная страница загружается
   - Откройте ваш Railway URL в браузере
   - Должна загрузиться главная страница SigmaVPN

2. Регистрация работает
   - Перейдите на страницу регистрации
   - Попробуйте создать аккаунт
   - Проверьте консоль браузера на ошибки

3. Вход работает
   - Попробуйте войти с созданным аккаунтом
   - Должны перейти в личный кабинет

4. API отвечает
   - Откройте DevTools (F12)
   - Перейдите на вкладку Network
   - Проверьте, что запросы идут на Railway URL

                          🆘 РЕШЕНИЕ ПРОБЛЕМ
════════════════════════════════════════════════════════════════════════════════

ПРОБЛЕМА: Git не установлен
РЕШЕНИЕ: Установите Git с https://git-scm.com/download/win

ПРОБЛЕМА: "fatal: not a git repository"
РЕШЕНИЕ: Выполните: git init

ПРОБЛЕМА: "fatal: 'origin' does not appear to be a 'git' repository"
РЕШЕНИЕ: Выполните: git remote add origin https://github.com/ВАШ_ЮЗЕРНЕЙМ/sigma.git

ПРОБЛЕМА: "Permission denied (publickey)"
РЕШЕНИЕ: Настройте SSH ключи на GitHub или используйте HTTPS с токеном

ПРОБЛЕМА: API не отвечает после развертывания
РЕШЕНИЕ: 
1. Проверьте, что API_URL обновлен в JavaScript файлах
2. Проверьте логи Railway (Railway Dashboard → Logs)
3. Убедитесь, что Railway развертывание завершено успешно

ПРОБЛЕМА: "ModuleNotFoundError: No module named 'flask'"
РЕШЕНИЕ: Railway автоматически установит зависимости из requirements.txt
